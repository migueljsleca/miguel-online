Signal Foundry reframes engineering documentation as a calmer product surface. The project explores how technical systems can feel precise without becoming cold, using restrained typography and direct structural cues to keep dense information readable.

### System thinking

The interface system focused on making documentation, field data, and operations views feel part of the same product family. Instead of treating engineering tools like separate utilities, the work aimed to create one consistent language across dashboards, specifications, and workflows.
